# 🚀 Setup Rápido - Registro de Usuários com Recompensa

## Configuração em 3 Passos

### 1️⃣ Executar Script de Configuração
```bash
./setup.sh
```

### 2️⃣ Seguir Tutorial Completo
Abra o arquivo: **[TUTORIAL_REMIX_DEPLOY.md](./TUTORIAL_REMIX_DEPLOY.md)**

### 3️⃣ Iniciar Aplicação
```bash
cd web
python3 -m http.server 8000
# Acesse: http://localhost:8000
```
