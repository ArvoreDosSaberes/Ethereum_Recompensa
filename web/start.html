🚀 DApp - Registro de Usuários com Recompensa

⚠️ Importante: Antes de iniciar, certifique-se de que o contrato foi compilado e implantado na rede local.

📋 Passo 1: Verificar Pré-requisitos

  • MetaMask instalado no navegador
  • Hardhat node rodando (http://127.0.0.1:8545)
  • Contrato implantado na rede local

⚙️ Passo 2: Configurar MetaMask

Adicione a rede local no MetaMask:

Network Name: Hardhat Local
RPC URL: http://127.0.0.1:8545
Chain ID: 1337
Currency Symbol: ETH

📍 Passo 3: Configurar Endereço do Contrato

Após fazer o deploy, configure o endereço:

localStorage.setItem('contractAddress', '0x...');

Substitua '0x...' pelo endereço real do contrato.

🎯 Passo 4: Iniciar a Aplicação

Escolha uma das opções abaixo para abrir a DApp:

🌐 Abrir DApp

Ou use um servidor local:

# Opção 1: Python
python -m http.server 3000

# Opção 2: PHP
php -S localhost:3000
